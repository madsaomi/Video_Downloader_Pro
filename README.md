# 🎬 Video Downloader Pro

Мощное десктопное приложение для скачивания видео с **YouTube**, **TikTok**, **Instagram**, **VK** и **1500+ других сайтов** с премиальным графическим интерфейсом.

## ✨ Возможности

### 🔍 Умный поиск
- **Встроенный YouTube-поиск** — введите текст вместо ссылки, и программа найдёт топ-10 видео прямо внутри приложения
- **Превью обложек** — результаты поиска отображаются с миниатюрами, названиями, авторами и длительностью
- **Быстрый выбор** — нажмите «Выбрать» напротив видео, и оно автоматически подготовится к скачиванию

### 📦 Пакетная загрузка
- **Batch Mode** — переключатель «Пакетная загрузка» превращает строку ввода в многострочное поле
- Вставьте несколько ссылок (по одной на строку) и скачайте всё разом
- Прогресс отображается для каждого видео: `[1/5] Скачивание...`

### 🎥 Скачивание
- **Выбор качества** — от 360p до 4K (2160p)
- **Только аудио** — извлечение в MP3 (320 kbps)
- **Обрезка видео** — указание начала и конца фрагмента (HH:MM:SS)
- **Субтитры** — скачивание встроенных или автосгенерированных субтитров
- **Превью** — предпросмотр обложки, названия и метаданных перед скачиванием

### 🍪 Авторизация
- **Cookies из файла** — поддержка `cookies.txt` для обхода возрастных ограничений (18+)
- **Cookies из браузера** — автоматическое извлечение cookies из Chrome, Firefox, Edge, Opera, Brave

### 🎨 Интерфейс
- **Премиум тёмная тема** — современный дизайн с закруглёнными элементами (16px)
- **Слайд-меню** — боковая панель скрывается/появляется по кнопке ☰ с плавной анимацией
- **Toast-уведомления** — неблокирующие всплывающие сообщения вместо системных окон
- **История** — автоматическое сохранение скачанных ссылок с быстрым доступом
- **Плейлисты** — полный просмотр содержимого YouTube-плейлистов

### 🔄 Автообновление
- **Обновление ядра yt-dlp** — кнопка в разделе «Настройки» обновляет загрузчик до последней версии
- Если YouTube изменил защиту — одно нажатие решает проблему

## 🛠 Требования

- **Python 3.10 — 3.13** (⚠️ Python 3.14 НЕ поддерживается — Pillow не имеет готовых сборок)
- **FFmpeg** — для склеивания видео+аудио в высоком качестве (1080p+) и конвертации в MP3
- **Deno** — JavaScript-runtime для расшифровки подписей YouTube

## 📦 Установка

### 1. Клонируйте или скачайте проект

```bash
git clone <url-репозитория>
cd Video_Downloader_Pro
```

### 2. Создайте виртуальное окружение

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Установите FFmpeg (обязательно)

```bash
winget install Gyan.FFmpeg
```

Или скачайте вручную с [ffmpeg.org](https://ffmpeg.org/download.html) и поместите `ffmpeg.exe` в папку проекта.

### 5. Установите Deno (обязательно для YouTube)

```bash
winget install DenoLand.Deno
```

## 📦 Портативная версия (.exe)

Можно собрать один `.exe` файл, который будет работать на **любом компьютере** без установки Python, FFmpeg или Deno.

### Сборка:
```bash
build.bat
```

Скрипт автоматически:
1. ✅ Скачает **FFmpeg** (если его нет в папке)
2. ✅ Скачает **Deno** (если его нет в папке)
3. ✅ Упакует **Python + все библиотеки** в один файл
4. ✅ Создаст `VideoDownloaderPro.exe`

### Что включено в .exe:
| Компонент | Описание |
|-----------|----------|
| Python | Встроен (PyInstaller) |
| yt-dlp | Встроен |
| CustomTkinter | Встроен |
| Pillow, requests | Встроены |
| FFmpeg | Встроен (если был в папке при сборке) |
| Deno | Встроен (если был в папке при сборке) |

> **Результат:** один файл `VideoDownloaderPro.exe` (~50-80 МБ), который можно скопировать на флешку и запустить на любом Windows ПК.

## 📖 Использование

### Скачивание по ссылке
1. Скопируйте URL видео → нажмите **«Вставить»**
2. Нажмите **«Поиск»** — загрузится превью и список форматов
3. Выберите качество → нажмите **«Скачать»**

### Поиск по YouTube
1. Введите текстовый запрос (например, `Смешные коты`) → нажмите **«Поиск»**
2. В окне результатов выберите видео с нужной обложкой → нажмите **«Выбрать»**
3. Выберите качество → нажмите **«Скачать»**

### Пакетная загрузка
1. Включите переключатель **«📦 Пакетная загрузка»**
2. Вставьте несколько ссылок (по одной на строку)
3. Нажмите **«Поиск»** → **«Скачать»** — все видео скачаются последовательно

## 🍪 Как использовать Cookies

Cookies нужны, если YouTube блокирует запросы или видео доступно только авторизованным пользователям.

### Экспорт cookies.txt из браузера:

1. Установите расширение:
   - Chrome/Edge: [Get cookies.txt LOCALLY](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)
   - Firefox: [cookies.txt](https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/)
2. Откройте **youtube.com** (убедитесь, что авторизованы)
3. Экспортируйте cookies → выберите файл в приложении через **«📂 Обзор»**

### Cookies из браузера (автоматически):
1. В разделе «Расширенные настройки» включите **«Использовать cookies из браузера»**
2. Выберите ваш браузер из списка (Chrome, Firefox, Edge, Opera, Brave)

## ⚠️ Частые проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| `Requested format is not available` | YouTube заблокировал IP | Подождите 5-10 мин, используйте cookies |
| `nsig extraction failed` | Нет Deno | `winget install DenoLand.Deno` |
| Видео без звука | Нет FFmpeg | `winget install Gyan.FFmpeg` |
| `ModuleNotFoundError` | Не установлены библиотеки | `pip install -r requirements.txt` |
| Маленькие разрешения | Устаревший yt-dlp | Настройки → «🔄 Обновить yt-dlp» |
| `Sign in to confirm you're not a bot` | Анти-бот защита | Экспортируйте cookies.txt |
