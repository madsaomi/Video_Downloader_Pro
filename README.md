# 🎬 Video Downloader Pro

Мощное приложение для скачивания видео с **YouTube**, **TikTok**, **Instagram**, **VK** и **1000+ других сайтов** с красивым графическим интерфейсом.

## ✨ Возможности

- 🎥 **Выбор качества** — от 144p до 4K (2160p)
- 🎵 **Только аудио** — скачивание в формате MP3 (320 kbps)
- 🖼 **Превью видео** — предпросмотр обложки и названия перед скачиванием
- 🍪 **Поддержка Cookies** — обход возрастных ограничений (18+) и защиты от ботов
- 🌙 **Тёмная тема** — современный премиум-дизайн
- 📋 **Кнопка «Вставить»** — быстрая вставка ссылки из буфера обмена

## 🌐 Поддерживаемые платформы (1500+ сайтов)

Приложение поддерживает скачивание видео и аудио с более чем **1500** различных платформ. В интерфейсе встроен динамический алфавитный каталог всех поддерживаемых сайтов.

**🔥 Самые популярные сервисы:**
- **Видеохостинги**: YouTube (Видео, Shorts, Плейлисты), Vimeo, Dailymotion, Rutube, Дзен, Яндекс.Видео, Bilibili, Niconico
- **Соцсети и Блоги**: TikTok, Likee, Instagram (Reels, IGTV, Посты), ВКонтакте (VK Видео, Клипы), Twitter / X, Facebook, Одноклассники, Reddit, Pinterest, Tumblr, LinkedIn, Patreon
- **Стримы и Игры**: Twitch (Воды, Клипы), Kick, AfreecaTV, Trovo
- **Музыка и Аудио**: SoundCloud, Bandcamp, Mixcloud, Spotify, Apple Music, Я.Музыка
- **Мессенджеры**: Telegram (публичные ссылки)
- **А также**: новостные сайты (BBC, CNN), образовательные платформы (Coursera), ТВ-каналы и почти любой сайт, где есть встроенный видеоплеер!

*(Внимание: в каталоге программы сайты с контентом 18+ скрыты фильтром, но само скачивание с них поддерживается ядром).*

## 🛠 Требования

- **Python 3.10 — 3.13** (⚠️ Python 3.14 НЕ поддерживается — Pillow не имеет готовых сборок)
- **FFmpeg** — нужен для склеивания видео+аудио в высоком качестве (1080p, 4K) и для конвертации в MP3
- **Deno** — JavaScript-runtime, необходимый для расшифровки подписей YouTube

## 📦 Установка

### 1. Клонируйте или скачайте проект

```bash
git clone <url-репозитория>
cd downloader
```

### 2. Создайте виртуальное окружение

```bash
python -m venv .venv
```

**Windows:**
```bash
.\.venv\Scripts\activate
```

### 3. Установите зависимости Python

```bash
pip install -r requirements.txt
```

### 4. Установите FFmpeg (обязательно)

FFmpeg нужен для скачивания видео в высоком качестве (1080p+) и извлечения аудио.

**Вариант A — через Winget (рекомендуется):**
```bash
winget install Gyan.FFmpeg
```

**Вариант B — вручную:**
1. Скачайте с [ffmpeg.org](https://ffmpeg.org/download.html)
2. Поместите `ffmpeg.exe` в папку проекта (рядом с `main.py`)

### 5. Установите Deno (обязательно для YouTube)

YouTube обновил защиту, и теперь `yt-dlp` не может расшифровать подписи видеопотоков без JavaScript-движка.

```bash
winget install DenoLand.Deno
```

Или скопируйте `deno.exe` в папку проекта (рядом с `main.py`).

## 🚀 Запуск

**Способ 1 — Двойной клик:**
Дважды кликните по файлу `start.bat`

**Способ 2 — Из терминала:**
```bash
.\.venv\Scripts\activate
python main.py
```

## 📖 Использование

1. **Вставьте ссылку** — скопируйте URL видео и нажмите кнопку «📋 Вставить»
2. **Нажмите «Получить информацию»** — приложение загрузит превью, название и список доступных разрешений
3. **Выберите качество** — из выпадающего списка (720p HD, 1080p Full HD, 4K, или Только аудио)
4. **Нажмите «Скачать»** — файл сохранится в выбранную папку (по умолчанию — `Downloads`)

## 🍪 Как использовать Cookies

Cookies нужны, если YouTube блокирует запросы (ошибка «защита от ботов») или видео доступно только для авторизованных пользователей (18+, подписки).

### Экспорт cookies.txt из браузера:

1. Установите расширение для браузера:
   - Chrome/Edge: [Get cookies.txt LOCALLY](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)
   - Firefox: [cookies.txt](https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/)
2. Откройте **youtube.com** и убедитесь, что вы авторизованы
3. Нажмите на иконку расширения → **Export** (скачается файл `cookies.txt`)
4. В приложении нажмите кнопку **«📂 Обзор»** и выберите скачанный файл

## ⚠️ Частые проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| `Requested format is not available` | YouTube заблокировал ваш IP | Подождите 5-10 мин, используйте cookies.txt |
| `nsig extraction failed` | Нет Deno | Установите: `winget install DenoLand.Deno` |
| Видео без звука | Нет FFmpeg | Установите: `winget install Gyan.FFmpeg` |
| `ModuleNotFoundError` | Не установлены библиотеки | `pip install -r requirements.txt` |
| Маленькие разрешения (45p, 27p) | Устаревший yt-dlp | `pip install --upgrade yt-dlp` |
| `Sign in to confirm you're not a bot` | Анти-бот защита YouTube | Экспортируйте cookies.txt из браузера |
